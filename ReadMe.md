# Welcome

This is the frontend application for the Movie Catalog of CRUD system provided in Blazor technology which integrates with the [MovieCatalog Web API](https://github.com/Chris2240/MovieCatalog_API) (backend). It provides a user-friendly interface to browse and manage your movie collection. To look on the frontend documentation please follow the link below:
